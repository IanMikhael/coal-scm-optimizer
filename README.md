# 🚀 Coal SCM Optimizer (Operational Control Tower)

Digital solution designed to optimize the Coal Supply Chain Management at **PT.XYZ Muara Enim**, based on the **SCOR (Supply Chain Operations Reference)** methodology.

### 🛠️ Tech Stack
- **Frontend:** React.js, Lucide Icons, Recharts (Bento Grid UI)
- **Backend:** Python Django, Django REST Framework
- **Admin UI:** Django Jazzmin (Modern Dashboard)

### 📊 Key Features
- **RKAB vs Realization Analysis:** Visual gap monitoring between production targets and actual results.
- **SCOR Metrics Alert:** Automated financial loss calculation based on POF (Perfect Order Fulfillment) & OFCT (Order Fulfillment Cycle Time) metrics.
- **Quality Dispute Management:** Digital module for handling product returns and laboratory evidence.
- **Global Price Index:** Real-time coal price tracking (ICI/Newcastle).
